# sql_analise_vendas

Este projeto foi criado para praticar análise de dados utilizando SQL.

O banco de dados simula um sistema de vendas contendo clientes, produtos, vendedores e vendas.

Tecnologias utilizadas:

SQL

Estrutura do projeto:

- criação do banco
- criação das tabelas
- inserção de dados
- consultas de análise

Principais análises realizadas:

- faturamento por mês
- produtos mais vendidos
- clientes que mais compraram
- ticket médio
